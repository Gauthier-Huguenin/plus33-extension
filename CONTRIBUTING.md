# Contributing

Thank you for taking the time to share ideas and improvements.

At this stage of the project, contributions are focused on **creating Issues**:
- Bug reports
- Feature requests

Please use the Issue templates provided:
- **Bug report** for any malfunction or incorrect behavior
- **Feature request** for new capabilities or improvements

Keep each Issue focused on a single topic, with enough detail to understand the context.

If you are reporting a bug:
- Provide reproduction steps
- Explain expected vs. actual behavior
- Include app/device information

If you are suggesting a feature:
- Describe the user problem
- Explain why your idea improves the experience

Clear Issues help guide development.  
Thank you for contributing to the project.
